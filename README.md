## アプリケーション名
「_Activer_」

## アプリケーション概要
### 自身の経歴や取り組みを整理し、日々の頑張りと共に記録・発信できるSNSサービス
自身の経歴や現在行っている活動などの情報を整理し、日々の頑張りと共に記録・発信することのできるSNSサービスです。  

主に、個人でのスキルアップや知識向上に力を入れている人や、個人で活動をしている人が、自身の活動を記録し、発信したい時に役立つサービスとなっています。  
パソコンとスマートフォンの両方に対応しており、場所を選ばず利用することが可能です。  

最大の特徴は、自身の情報や投稿を整理しやすいことです。  
マイページでは、自身の簡単な情報を記載できるプロフィール作成機能のほか、自身の経歴や活動の情報を整理して表示できるメインコンテンツ機能や、自身の収益活動の情報を整理して表示できるストアコンテンツ機能があり、自身の情報を細かくまとめる表示することができます。  
さらに、自身でカテゴリーを作成し、投稿とメインコンテンツに設定することもできるため、投稿をまとめたり、投稿とメインコンテンツを紐づけたりすることも可能です。  
その他にも、投稿時にチェックを入れるだけで、特に見てもらいたい「ビッグポスト」として作成し、それらをまとめて見てもらうこともできます。  

また、他のユーザーとのつながりも作りやすくなっています。  
いいね機能やフォロー機能があるのはもちろん、ユーザーごとに設定できるユーザータグと投稿ごとに設定できる投稿タグの2種類のタグを利用することができるため、投稿やユーザー同士の関連性構築と内容理解により、容易に他のユーザーとのつながりを作ることができます。  

さらに、これらの投稿・メインコンテンツ・ストアコンテンツ・ユーザーを、タグを含めた複数キーワードでの検索をすることもでき、情報収集やユーザー検索もしやすくなっています。

## 利用方法
### ログイン前
ログイン前には、トップページのみ見ることが可能です。  
新規登録画面にてアカウントを作成し、ログインすることで、全ての機能を利用することができます。  

### おすすめの初期設定
また、登録したら、先にユーザータグとマイカテゴリーを設定することをお勧めします。  
　ユーザータグ・・・自身の投稿やマイページを見てもらったときに、瞬時に自身の特徴を知ってもらうことができるます。  
　マイカテゴリー・・・マイカテゴリーを設定しておくことで、投稿をまとめ、投稿とメインコンテンツを関連付けることができます。　　
どちらも1分ほどで設定ができるため、ぜひ試してみてください。 

### マイページの充実化
マイページでは、メインコンテンツとストアコンテンツを作成することができます。どちらも早めに作成することをお勧めしますが、お好きなタイミングで作成、編集が可能です。  
　メインコンテンツ・・・自身の活動内容をまとめ、紹介することができます。  
　ストアコンテンツ・・・収益活動への導線をまとめることができます。 

### その後
これらの設定が終われば、あとは適宜つぶやきや頑張りの過程を投稿して、好きなように活動するだけです 。  
イベントの開催や、情報収集などに使うのも良いかもしれません。  

当初、このアプリは活動の記録・発信のために制作しましたが、それ以外にも、私はこのアプリを通して、使ったすべての人に交流を増やしてもらいたいと思っています。  
人とのつながりは、きっと、今行っている活動を始め、人生全体をより良いものにしてくれると思います。  
Activerがたくさんの人の活動と人生をより良いものにすることを祈っています。

## 目指した課題解決
現在は、情報発信ができるSNSや個人で活動ができるプラットフォームが多くあり、それらを使って情報発信や個人で活動をするも多くいます。  
そして、私は以前からこれらを使う中で、個人の経歴や活動内容の情報が分散してしまっていることに問題を感じていました。  

例えば、SNSのアカウントは検索性や内容の一貫性という点から、活動や用途に応じて複数作成する場合が多く、また、プラットフォームに関してもSNSと同じように、その人が提供したい価値に合わせて使い分ける場合が多いです。  
そのため、個人が幅広く活動すればするほど、情報が分散してしまい、個人全体として見てもらうことが難しくなってしまいます。  

そして、私はそのように頑張ってきたことの一部分しか見てもらえないことが、とてももったいないと感じました。個人で活動するにしても、就職や転職をするにしても、その人が行ってきたことや今行っていること全体を見てもらい、評価され、信用されるべきです。  

このことから、私はこの問題を解決するために、自身の経歴や取り組みを整理し、日々の頑張りと共に記録・発信できるSNSサービス「Activer」を開発することに決めました。
